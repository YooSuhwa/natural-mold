## Codebase Patterns
- Backend: FastAPI + SQLAlchemy async + Alembic (Python 3.12, uv)
- Frontend: Next.js 16 + React 19 + TailwindCSS v4 + shadcn/ui (pnpm)
- AI Runtime: LangChain 1.x + LangGraph 1.x + DeepAgents (create_deep_agent)
- DB: PostgreSQL 16 (docker-compose)
- Router→Service→Model 패턴 (async/await, Depends(get_db))
- 프론트: TanStack Query (서버) + Jotai (클라이언트) + SSE 스트리밍
- i18n: next-intl, 아이콘: lucide-react, 토스트: sonner
- 현재 인증 없음 — get_current_user는 Mock User 반환
