## 인터페이스 계약

### BuilderState (TypedDict)
```python
class BuilderState(TypedDict):
    messages: Annotated[list[BaseMessage], add_messages]
    todos: list[PhaseTodo]              # 8개, 매 노드 진입 시 갱신
    user_request: str                    # 첫 메시지
    intent: dict | None                  # Phase 2 결과
    tools: list[dict]                    # Phase 3 결과
    middlewares: list[dict]              # Phase 4 결과
    system_prompt: str | None            # Phase 5 결과
    image_url: str | None                # Phase 6 결과
    draft_config: dict | None            # Phase 7 결과
    current_phase: int                   # 1~8
    last_revision_message: str | None    # phase 3/4/5/6 수정요청 시 LLM에 전달
```

### Interrupt Payload (3종)
- ask_user: `{"type": "ask_user", "question": str, "options": list[str]}`
- approval: `{"type": "approval", "data": dict, "summary": str, "allow_revision": bool}`
- choice: `{"type": "choice", "title": str, "options": list[str], "context": dict}`

### Resume Payload
- ask_user 응답: `str` (자유 텍스트 또는 옵션)
- approval 응답: `{"approved": bool, "revision_message": str | None}`
- choice 응답: `{"choice": str, "extra": dict}` (예: image regenerate prompt 편집)

## Codebase Patterns

- Backend: Router → Service → Model, async SQLAlchemy 2.0, select() 구문
- 테스트: `backend/tests/test_<resource>.py`, aiosqlite in-memory conftest
- LangGraph checkpointer: `get_checkpointer()` (PostgresSaver), `thread_id = builder_session_id`
- 새 모듈은 `backend/app/agent_runtime/builder_v3/`에 작성
- Frontend: `lib/api/<domain>.ts` + `lib/hooks/use-<domain>.ts` + `lib/types/index.ts`
- Tool UI는 `components/chat/tool-ui/*.tsx`에 추가 + `lib/chat/tool-ui-registry.ts`에 등록
