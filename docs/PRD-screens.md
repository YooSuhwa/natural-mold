### 7. 화면 구성

#### 7.1 화면 목록

| 화면명 | 설명 | 진입 경로 |
|--------|------|----------|
| 대시보드 | 내 에이전트 목록 + 사용량 요약 | / (메인) |
| 에이전트 생성 — 방식 선택 | 대화로 / 템플릿으로 만들기 | 대시보드 → [+ 새 에이전트] |
| 에이전트 생성 — 대화형 | AI와 대화하며 에이전트 구성 | 방식 선택 → 대화로 만들기 |
| 에이전트 생성 — 템플릿 | 템플릿 탐색 및 선택 | 방식 선택 → 템플릿으로 만들기 |
| 에이전트 채팅 | 에이전트와 대화 | 대시보드 → 에이전트 카드 클릭 |
| 에이전트 설정 | 이름, 프롬프트, 도구, 모델 수정 | 채팅 → 설정 아이콘 |
| 도구 관리 | 도구 목록, MCP/커스텀 도구 등록 | 사이드바 → 도구 관리 |
| 모델 관리 | 모델 목록, 커스텀 모델 등록 | 사이드바 → 모델 관리 |
| 사용량 대시보드 | 에이전트별 토큰·비용 | 사이드바 → 사용량 |

#### 7.2 화면 흐름도

```
[대시보드] ──[+ 새 에이전트]──> [방식 선택]
   │                              ├──> [대화형 생성] ──확정──> [채팅]
   │                              └──> [템플릿 선택] ──생성──> [채팅]
   │
   ├──[에이전트 카드]──> [채팅] ──[설정]──> [에이전트 설정]
   ├──[사이드바: 도구]──> [도구 관리] ──[+ 추가]──> [MCP 서버 / 직접 정의]
   ├──[사이드바: 모델]──> [모델 관리]
   └──[사이드바: 사용량]──> [사용량 대시보드]
```
